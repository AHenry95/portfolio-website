Chat App

Chat App is a real-time messaging application built with React Native and Expo, demonstrating modern mobile development practices & capabilities. The app also features Firebase integration for real-time data synchronization, secure anonymous authentication, and cloud storage for media files. The app is designed to work seamlessly on both iOS and Android devices using Expo Go. Users can send text messages, images (both from their device library or taken with their camera), and they can send their location, which will be displayed in a rendered map in the chat. The application can detect if the user's device is connected to the internet or not, and if not will display cached messages until the device is connected to the internet again.

The Chat App Welcome Screen, where users can enter their username, select a background color, and move to the chat view.

A demonstration of the Chat App, including sending images and location.

Features

  • Real Time Messaging
  • Firebase/Firestore Integration
  • Anonymous User Authentication
  • Image Sharing from Camera or Photo Library
  • Location Sharing Featuring Interactive Maps
  • Message Caching + Offline Cached Message Display
  • Cross-Platform (iOS/Android) Support
