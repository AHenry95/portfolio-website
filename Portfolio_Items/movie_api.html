Movie API

This project is a RESTful API built with Node.js and Express that provides movie information, user authentication, and user profile management. This backend service powers movie applications by serving data about films, directors, genres, and actors while managing user accounts and personalized favorite lists. Working on this project was my first time creating an API, and using tools such as Node.js and Express to create the server. Not only was this very informational for working on similar backend projects, it also helped me understand even better how to interact with APIs, as it gave me an even clearer idea of how they actually work.

The movie API's home page, displaying documentation.

Features

  • Secure, JWT-based User Authentication
  • Movie Database
  • User Profiles
  • Data Validation
  • RESTful Design

Technologies and Packages Used

  • Node.js
  • Express.js
  • MongoDB with Mongoose
  • Passport.js
  • JWT
  • bcrypt
  • Express Validator
  • CORS
  • Morgan
  • Heroku
