Pokédex

This project is a pokedex that allows users to view information about any of the 151 "first generation" Pokémon. In its current iteration, the app displays a basic list of buttons, each of which contains the name of a Pokémon, ordered by their number (not displayed on the button). When clicked, each button displays the Pokémon's name, their sprite image, and their height in meters. All of the Pokémon data is fetched from the PokéAPI. This project was my first time working with an API to fetch data to be displayed, skills which I have used throughout all of the other projects I have worked on. Much of the styling of the app is provided via Bootstrap, which was my first time working with an external library to provide styling for an app.

I intend to return to this project very soon and improve the visual display and increase the number of features offered to users. You can view more details about planned improvements in the project's README. Please feel free to provide comments or suggest features to be added via my contact form. Additional insights are always welcome and much appreciated!
