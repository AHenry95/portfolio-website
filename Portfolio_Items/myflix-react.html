myFlix - React

myFlix is a client React application that allows users to browse and discover information about their favorite movies. The application provides access to data to movies accessed via the Movie API, including titles, directors, actors, and release years for each movie. Additionally, users can view information about each director and genre included in the database. myFlix is built with modern web technologies, and features a responsive design powered by React Bootstrap. Users can create and manage personal accounts, including the ability to manage a list of favorite movies, that are visible on their profile. The application allows for the filtering of movies displayed to the user in the main movie view, allowing users to easily view all movies that meet their chosen criteria (i.e., director, genre).

The MainView component, displaying all the movies from the database, with not filtering applied The MovieView component, displaying detailed information about a movie The UserView component, displaying information about a user.

Features

  • User Authentication (Registration/Login)
  • Movie Card Display
  • Movie Search Function
  • Movie Filtering Function
  • Detailed Information Available for Each Movie
  • User Favorites Management
  • User Profile Management
  • Responsive Design for All Devices

Technologies and Packages Used

  • React 19.1.0
  • React DOM 19.1.0
  • React Router 7.7.0
  • React Bootstrap 5.3.7
  • Parcel/Transformer-SASS 2.15.4
  • PropTypes 15.8.1
  • Movie API
