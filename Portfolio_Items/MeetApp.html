Meet App

The Meet App is a serverless, progressive web application (PWA) built with React that enables users to discover and explore upcoming events in their area. The application used the Google Calender API to fetch real-time event data and presents it through an intuitive, responsive interface that included interactive data visualizations. Built using test-driven devlopment (TDD) methodology, the app combines modern web technologies with serverless architecture to deliver a fast and reliable user experience. Users can search for events by city, choose the number of events displayed, and view detailed event information via the expandanble event cards. The application features two dynamic charts: a scatterplot that displays the distribution of listed events across cities and a pie chart that displays the event genres by percentage. The app uses AWS Lambda functions for authentication and API calls, implements Google OAuth 2 for secure access, and includes comprehensive PWA features such as offline access, home screen installation, and service worker caching.
