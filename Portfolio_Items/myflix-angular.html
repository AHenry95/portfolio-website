myFlix - Angular

myFlix is a client application created with Angular, which accesses and displays movie information from Movie API. The app displays detailed information about each movie in the accessed database, such as including information about each movie's director, genre, and plot. Google's Material Design is used to create an attractive, responsive, and intuitve user interface. Users can create and manage their accounts, which are secured using JWT authentication. Users, as part of their account, can manage a list of their favorite movies, which are displayed on their profile page, and are kept up to date throughout the app.

myFlix's main view, displaying all movies from the database. myFlix's user view, displaying the user's account information, as well as their favorite movies.

Features

  • User Authentication (Registration/Login)
  • Movie Card Display
  • Detailed Information Available for Each Movie
  • User Favorites Management
  • User Profile Management
  • Responsive Design for All Devices
